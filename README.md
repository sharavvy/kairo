# KAIRO - учебный сайт ресторана

Исправленная версия сайта: русские тексты сохранены в UTF-8, внешние шрифты не используются, крупные заголовки ограничены по ширине, мобильная вёрстка переработана. Телефон убран из верхней навигации. Вход и регистрация объединены в одну кнопку «Аккаунт», бронирование открывается в модальном окне, а форма отзыва находится под каруселью отзывов.

## Запуск

1. Распакуйте архив.
2. Откройте `index.html` в браузере.

## Что реализовано

- главная страница;
- бронирование стола через всплывающее окно;
- раздел «О ресторане»;
- меню с категориями и поиском;
- мероприятия;
- единая кнопка «Аккаунт» в навигации с вкладками входа и регистрации;
- отзывы с тремя примерами, стрелками для листания и формой отправки под отзывами;
- контакты;
- демо-админка для добавления и удаления блюд;
- адаптивная вёрстка для мобильных устройств.

## Важно

Это учебная frontend-версия. Данные сохраняются в `localStorage`. Для реального сайта нужна серверная часть и безопасное хранение паролей.


В каждой категории меню добавлено по 5 блюд. Вкладка "Рекомендованные" показывает 5 выбранных позиций.



## Фотографии
